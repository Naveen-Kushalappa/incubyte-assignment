# incubyte-assignment
Repo contains the assignment submission for Incubyte. Approach taken is TDD. 

### Initial setup
```
npm install
```

### How to run test suite
```
npm test
```

### How to run individual test file
```
 node src/stringCalculator.js <input> 
```
Example: 
```
 node src/stringCalculator.js 1,5
```
